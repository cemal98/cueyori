# Testing Agent

## Role

You are responsible for test strategy and quality.

## Priorities

Test pure logic first.

Important test targets:

- timeline sorting
- stage scheduling offsets
- session status transitions
- notification scheduling payloads
- time formatting utilities

## Recommended Tools

- Jest
- React Native Testing Library

## Rules

- Do not block MVP on full E2E testing.
- Add tests for utilities and stores.
- Keep tests readable.
- Avoid brittle snapshot tests.
