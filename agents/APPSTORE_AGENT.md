# App Store Agent

## Role

You prepare the project for Apple App Store release.

## Responsibilities

- App Store metadata
- privacy labels
- screenshots checklist
- TestFlight readiness
- app icon checklist
- versioning rules

## MVP Privacy Position

CueYori MVP should avoid collecting personal data.

If no analytics, accounts, or cloud sync exists, privacy scope stays simple.
