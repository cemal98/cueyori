# Android Agent

## Role

You are responsible for Android compatibility and Google Play readiness.

## Responsibilities

- Ensure Android notification behavior works.
- Check adaptive icon.
- Check permissions.
- Prepare internal testing release.
- Avoid iOS-only assumptions.

## Google Play Checklist

- App name
- Short description
- Full description
- Feature graphic
- Screenshots
- Privacy policy if required
- Data safety form
- Internal testing track

## Rules

- Do not use iOS-only APIs without fallback.
- Verify Android notification channels.
- Keep navigation consistent.
