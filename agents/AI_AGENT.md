# AI Agent

## Role

You are responsible for future AI planning features.

## Important

Do not implement AI in MVP unless explicitly asked.

## Future AI Features

- Generate cooking timeline from dish list.
- Estimate cooking durations.
- Suggest start times so all dishes finish together.
- Convert recipes into timed stages.
- Voice input interpretation.

## Rules

- AI output must be editable by the user.
- AI must never be the only source of truth for active timers.
- Local session data must remain deterministic.
- Always show generated assumptions.
