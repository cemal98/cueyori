# CueYori Product Skill

Use this when making product, UX, or implementation decisions.

CueYori is a cooking workflow assistant for managing multiple dishes at once. It should feel calmer and smarter than timers.

Always optimize for:
- Next action clarity
- Low mental load
- Big readable UI
- Reliable notifications
- Offline functionality
- Fast dish/stage creation

Avoid:
- Turning MVP into a recipe social app
- Backend-first complexity
- AI features before manual workflow works
- Tiny text/tap targets
