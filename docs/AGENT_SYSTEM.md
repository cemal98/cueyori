# CueYori Agent System

This project includes a role-based AI agent system for building CueYori with
Codex, Cursor, Claude Code, or similar tools.

## How to Use

1. Open the project in an AI coding environment.
2. Read `AGENTS.md`.
3. Read every file in `agents/`, `docs/`, and `skills/`.
4. Start from `prompts/CODEX_FIRST_MESSAGE.md` when beginning a new agent
   session.

## Recommended Workflow

Do not ask an agent to build the entire app at once. Use this sequence:

1. Architecture
2. Design system
3. Navigation
4. Cooking session store
5. Dish creation
6. Timeline
7. Local notifications
8. Polish
9. EAS deployment

## Important

This system exists to keep CueYori product decisions, engineering constraints,
and release readiness consistent across tools.
